Connect your ESP32-based Sense360 device via USB cable to begin flashing firmware.

Connection Requirements

  • Chrome, Edge, or Opera browser
  • USB cable connected to device
  • Device in bootloader mode

About This Project

Features

  • • One-click firmware installation
  • • Multiple firmware version support
  • • Real-time flashing progress
  • • Built-in Wi-Fi setup guide
  • • Comprehensive troubleshooting

Supported Hardware

  • • ESP32-S3-WROOM-1-N16R8
  • • Hi-Link LD2412 Radar Module
  • • MiCS-4514 Gas Sensor
  • • Multiple air quality sensors
  • • WS2812B LEDs & PWM Fan
